# Gemology Sales



Single-page sales site for the **Gemology Journey** ebook (Cabo Da Nau GemStones), including bonus section and YouTube authority block.



- **Main entry:** open [`index.html`](./index.html) in your browser (same content as `gemology-sales.html`).

- **Alternate filename:** [`gemology-sales.html`](./gemology-sales.html) (duplicate of `index.html`).



Checkout URL is set in the page script (`HOTMART_CHECKOUT_URL`).



You do not need Node.js or `npm` to view it locally.
